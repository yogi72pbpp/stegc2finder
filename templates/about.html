📌 Project Overview

This system detects hidden information embedded inside digital images using Noise Residual Analysis and Machine Learning. Steganography hides secret data inside images imperceptibly — making it a serious cybersecurity threat when used maliciously to deliver malware payloads. Our system uses NumPy and OpenCV to compute multi-scale Gaussian noise residuals, extracts 16 statistical features, and classifies images using a trained Random Forest model.
